# Software Challenges 🚀

Organización personal donde centralizo pruebas técnicas, retos de programación y proyectos de práctica con enfoque profesional.

Cada proyecto busca simular escenarios reales de desarrollo, aplicando buenas prácticas, arquitectura limpia, despliegue cloud y documentación clara.

## Proyectos

* ☁️ **[Banking Microservices](https://github.com/Software-Challenges/SoftBanking-Challenge.git)** - Arquitectura de microservicios independientes y dockerizados que gestionan la información de un sistema bancario.
* 🏦 **Investment Funds API** *(WIP)* - API REST para gestión de fondos de inversión. (Java + Spring Boot) Arquitectura Hexagonal + AWS.
